# 2026 超浪漫跨年倒计时项目

## 1. 项目概述
- **项目名称**：2026 元旦跨年超浪漫倒计时项目、绚丽的烟火背景
- **目标受众**：追求仪式感、热衷在社交媒体（抖音/朋友圈/小红书）分享的年轻人
- **核心价值**：用震撼的粒子视听体验，把个人年度回忆照片与新年倒计时深度结合，打造浪漫、唯美、酷炫的跨年仪式。
- **视觉风格**：金色点缀 、红包雨、粒子流光 、空间旋转展示照片，手势抓取随机弹出照片，在照片下面弹出网络热门祝福语、绚丽的烟火背景

## 3. 阶段一：梦幻入口（照片上传）

### 3.1 视觉设计
| 元素 | 规格说明 |
|------|----------|
| **背景** | 深空渐变 `radial-gradient(ellipse at center, #1a1a2e 0%, #0a0a0f 70%, #000 100%)` + 浮动星光粒子 +浪漫流星雨|
| **玻璃卡片** | `backdrop-filter: blur(20px)`，边框 `1px solid rgba(255,215,0,0.2)`，内发光效果 |
| **主标题** | 渐变色 `linear-gradient(135deg, #FFD700, #FFA500)`，文字发光 `text-shadow: 0 0 30px rgba(255,215,0,0.5)` |
| **按钮** | 金色渐变背景，hover 时发光增强 `box-shadow: 0 0 40px rgba(255,215,0,0.6)`，pulse 呼吸动画 |

### 3.2 交互规格
- **上传区域**：拖拽高亮反馈（金色边框 pulse）
- **照片预览**：3x3 网格布局，缩放进入动画 `scale(0) → scale(1)`，带弹性缓动
- **照片数量**：5-10 张，实时计数显示
- **开始按钮**：5 张照片启用，金色渐变 pulse 效果

## 4. 阶段二：粒子倒计时（核心亮点）

### 4.1 背景氛围
```
┌─────────────────────────────────────────────────┐
│  深色背景 + 环境粒子流（持续漂浮的彩色星尘）      │
│  星尘颜色：金色 #FFD700、紫色 #A855F7、青色 #06B6D4 │
│  粒子数量：缓慢随机运动             │
└─────────────────────────────────────────────────┘
```

### 4.2 数字粒子特效
| 特性 | 实现规格 |
|------|----------|
| **粒子数量** | 每个数字 1500-2000 个粒子 |
| **粒子颜色** | 金色系（#FFD700、#FFA500、#FFEC8B）随机 |
| **粒子大小** | 2-3px，带 glow 发光效果 `shadowBlur: 8` |
| **运动模式** | 旧粒子"爆炸扩散" → 新粒子"汇聚成形" |
| **缓动效果** | elastic 弹性缓动，0.8-1s 过渡 |
| **发光效果** | 每个粒子绘制时叠加 `globalCompositeOperation: 'lighter'` |

### 4.3 数字切换动画（关键）
```
1. 当前数字粒子向四周"爆炸"扩散（速度 15-25px/frame）
2. 0.2s 后新数字粒子从中心汇聚
3. 汇聚过程带弹性缓动 easeOutElastic
4. 粒子到达目标位置后轻微"呼吸"震动
```

### 4.4 倒计时节奏
| 秒数 | 效果 |
|------|------|
| 10-4 | 标准粒子动画，环境粒子缓慢漂浮 |
| 3-1 | **紧张模式**：粒子数量 +30%，爆炸速度 +50%，背景红色边缘光晕 |
| 0 | **终极爆发**：屏幕中央白色闪光 0.3s → 粒子四散 → 转场阶段三 |

### 4.5 Canvas 渲染优化
```javascript
// 关键技术点
- 使用 offscreen canvas 预渲染数字形状
- 粒子对象池复用，避免 GC
- requestAnimationFrame + 时间差计算保证 60fps
- 拖尾效果：半透明覆盖 `rgba(10,10,15,0.15)`
```

## 5. 阶段三：盛大终章（3D + 烟花）

### 5.1 3D 照片旋转木马
| 特性 | 规格 |
|------|------|
| **布局** | 照片环形阵列，半径 `R = (width/2) / tan(π/N)` |
| **动画** | 无限旋转 `rotateY(0deg) → rotateY(360deg)`，周期 15-25s（根据照片数） |
| **照片样式** | 圆角 12px，金色边框 3px，边框发光 `box-shadow: 0 0 30px rgba(255,215,0,0.6)` |
| **边框动画** | 2s 呼吸动画 `borderGlow`，透明度 0.3 ↔ 0.8 |
| **交互** | 点击/触摸暂停旋转，再点击恢复 |

### 5.2 烟花系统
| 烟花类型 | 规格 |
|----------|------|
| **上升** | 从底部随机位置发射，速度 8-15px/frame，带尾迹 |
| **爆炸** | 3 种模式随机：<br>• 球形爆炸（60-100 粒子）<br>• 心形爆炸（数学方程）<br>• 螺旋爆炸（角速度递增） |
| **粒子** | HSL 随机色，重力 0.05-0.1，摩擦 0.95-0.98，透明度衰减 0.01-0.02 |
| **数量** | 同时 1-3 个烟花循环发射，间隔 0.5-2s |
| **地面反射** | 爆炸点在 y=0.7H 以下时，下方绘制半透明反射 |
